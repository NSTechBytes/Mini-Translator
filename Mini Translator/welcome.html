Welcome to Mini Translator Extension!

Thank you for installing the Mini Translator Extension for Chrome! This extension is designed to help you easily translate selected text and listen to translations using text-to-speech technology. Whether you're browsing the web or reading content in a different language, Mini Translator simplifies your experience.

Features

  • Text Translation: Select any text on a webpage and instantly translate it to your preferred language.
  • Popup Translation: Input text in the popup and translate it to various languages with a single click.
  • Text-to-Speech (TTS): Listen to the translated text using TTS functionality, with voices that match the selected language.
  • Language Selection: Choose from a wide range of languages and the extension will remember your selection for future translations.
  • Customizable Theme: Toggle between light and dark mode for a better user experience based on your preferences.
